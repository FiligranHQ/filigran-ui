# Select Example

<Select>
  <SelectTrigger className="w-[180px]">
    <SelectValue placeholder="Theme" />
  </SelectTrigger>
  <SelectContent>
    <SelectItem value="light">Light</SelectItem>
    <SelectItem value="dark">Dark</SelectItem>
    <SelectItem value="system">System</SelectItem>
  </SelectContent>
</Select>

## How to use this select

### Props

| Name            | Type                       | Default | Description                                                                                          |
| --------------- | -------------------------- | ------- | ---------------------------------------------------------------------------------------------------- |
| **asChild**     | _boolean_                  | false   | Change the default rendered element for the one passed as a child, merging their props and behavior. |
| **placeholder** | _string_ (for SelectValue) | -       | Text that appears in the form control when it has no value set.                                      |
| **value**       | _string_ (for SelectItem)  | -       | The value of the field if it is selected.                                                            |
