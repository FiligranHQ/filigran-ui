## How to use this separator

### Props

No props in it ! It's a hook !

### Playground

_Import from filigran-ui :_

Import \{useToast\} from 'filigran-ui'

_Use the hook in the component :_

const \{toast\} = useToast()

_You can use it in your component :_

<ReactLiveDisplay
  displayError={false}
  codeExample={`<Button
        onClick={() => {
          toast({
            title: 'A title',
            description: 'My description',
            variant: 'destructive',
            action: <Button variant="outline">Click me</Button>
          })
        }}>
        Show Toast
      </Button>`}
/>
