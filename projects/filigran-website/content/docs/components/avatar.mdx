## How to use this multiselect
### Props

| Name               | Type                                | Default     | Description                                                                                |
| -------------      | --------                            | ----------- | --------------------------------------------------------------------------------------     |
| **src**            | *String*                            | ''          | On the AvatarImage, set the image that should be displayed :)                              |
