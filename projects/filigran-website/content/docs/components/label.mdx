# Label example

<div>
  <Label htmlFor="terms">Accept terms and conditions</Label>
</div>

_This component is based on the native label element, it will automatically apply the correct labelling when wrapping controls or using the htmlFor attribute. For your own custom controls to work correctly, ensure they use native elements such as button or input as a base._

## How to use this label

### Props

| Name        | Type      | Default | Description                                                                                          |
| ----------- | --------- | ------- | ---------------------------------------------------------------------------------------------------- |
| **asChild** | _boolean_ | false   | Change the default rendered element for the one passed as a child, merging their props and behavior. |
| **htmlFor** | _string_  | -       | The id of the element the label is associated with.                                                  |
